在 Microsoft Word 中，“固定代码域”通常指通过域代码（如 `{ DATE }`、`{ PAGE }` 等）实现动态内容的字段。以下是与域操作相关的所有常用快捷键，适用于 Windows 版 Word（Mac 版略有不同）：

域操作快捷键汇总

- 插入空域：`Ctrl + F9`  
  插入一对灰色底纹的大括号 `{}`，光标自动定位其中，可输入域代码（如 `DATE`、`PAGE`）。

- 切换单个域的代码与结果：`Shift + F9`  
  选中某个域后按此组合键，可在显示域代码和域结果之间切换。

- 切换整个文档所有域的代码与结果：`Alt + F9`  
  快速显示或隐藏文档中所有域的代码。

- 更新选中域：`F9`  
  更新当前选中的域内容（如刷新页码、日期等）。

- 锁定域（防止更新）：`Ctrl + F11`  
  锁定后，即使按 F9 或打印也不会更新该域内容。

- 解锁域：`Ctrl + Shift + F11`  
  解除锁定，使域恢复可更新状态。

- 将域结果转换为静态文本：`Ctrl + Shift + F9`  
  移除域代码，保留当前显示结果为普通文本。

- 定位到下一个域：`F11`  
  在多个域间快速跳转到下一个。

- 定位到上一个域：`Shift + F11`  
  向前跳转到上一个域。

> ⚠️ 注意：  
> - 手动输入的大括号 `{}` 不能作为有效域，必须使用 `Ctrl + F9` 插入。  
> - Mac 版 Word 的部分快捷键不同，例如切换所有域代码使用 `fn + Option + F9`。

如需进一步了解具体域（如 `TOC`、`SEQ`、`IF` 等）的语法和用法，可参考 [Microsoft 官方支持文档](https://support.microsoft.com/zh-cn/office/word-中的域代码列表-1ad6d91a-55a7-4a8d-b535-cf7888659a51)。
