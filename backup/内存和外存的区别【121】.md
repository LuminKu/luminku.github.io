电脑的内存和外存在不同语境、行业或技术文档中存在多种称呼，以下是基于权威公开资料整理的常见别名：

内存的常见名称或称呼
- 主存（Main Memory）：强调其作为CPU直接访问的主要存储区域 。
- 运行内存：常用于手机、电脑等设备的用户界面，指当前运行程序所用的内存 。
- RAM（Random Access Memory）：特指随机存取存储器，是内存的主体部分 。
- 工作内存：因其用于存放正在执行的程序和数据而得名 。
- 内部存储器：与“外部存储器”（外存）相对的术语 。

外存的常见名称或称呼
- 辅存（Auxiliary Storage）：与“主存”对应，表示辅助主存的存储设备 。
- 存储空间：日常用语中常被误称为“内存”，尤其在手机营销中（如“128G大内存”）。
- 硬盘 / 固态硬盘 / U盘 / 光盘 / 磁带：具体外存设备的名称 。
- 长期存储 / 永久存储：强调其断电后数据不丢失的特性 。
- Storage：英文术语，在安卓等系统中常指“Internal Storage”或“External Storage”。

> ⚠️ 注意：在手机行业，“内存”一词常被误用于指代外存（如将UFS闪存称为“内存”），这与计算机科学中的标准定义不符 。严格来说，内存 = RAM + ROM，而外存 = 硬盘、SSD、U盘等。
