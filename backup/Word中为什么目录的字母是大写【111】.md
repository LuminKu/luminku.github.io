Word中目录字母变成大写，通常不是因为文档内容本身被修改，而是由于样式设置或自动格式功能导致的。以下是几种常见原因及对应的解决方法：

---

主要原因

- 目录样式启用了“全部大写”或“小型大写字母”效果  
  Word的目录样式（如“目录1”“目录2”）可能继承了标题样式中的字体格式设置，其中勾选了“全部大写字母”或“小型大写字母”。

- 自动更正规则触发  
  Word的“自动更正”功能中若启用了“句首字母大写”或“全部单词大写”等选项，也可能影响粘贴或生成的目录内容。

- 粘贴时带入了原始格式  
  直接使用 `Ctrl+V` 粘贴内容会保留源格式，包括大小写设置；建议使用“只保留文本”粘贴。

---

解决方法

1. 修改目录样式，关闭“全部大写”  
   - 右键点击目录中的任意一项（如“目录1”）→ 选择 “修改”。  
   - 在弹出窗口中点击左下角 “格式” → 选择 “字体”。  
   - 在 “效果” 区域，取消勾选 “全部大写字母” 和 “小型大写字母”。  
   - 点击 “确定” → 再次点击 “确定” 关闭样式窗口。  
   - 最后，更新目录：右键目录 → 选择 “更新域” → “更新整个目录”。

2. 使用快捷键切换大小写  
   - 选中目录内容（或整个目录）。  
   - 按下 `Shift + F3`，可在以下状态间循环切换：  
     - 全部大写  
     - 全部小写  
     - 首字母大写  
   此操作立即生效，无需修改样式。

3. 检查并调整自动更正设置（可选）  
   - 文件 → 选项 → 校对 → “自动更正选项”。  
   - 在“键入时自动套用格式”和“自动更正”选项卡中，检查是否勾选了不必要的全大写规则，酌情取消。

4. 粘贴时使用“只保留文本”  
   - 复制内容后，右键粘贴 → 选择 “只保留文本”（图标为“A”），或使用快捷键 `Ctrl + Alt + V` → 选择“无格式文本”。

---

通过上述任一方法，通常可恢复目录中英文的正常大小写格式。若问题仍存在，建议检查是否使用了特定机构提供的论文模板，其内置样式可能强制转换大小写。
