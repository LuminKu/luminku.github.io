## 查看网络端口占用

在 Windows 系统中，可以通过命令行工具快速查看网络端口的占用情况，并找到对应的进程。

示例步骤

1. 查看所有端口使用情况

打开命令提示符（以管理员身份运行），输入以下命令：

```bash
netstat -ano
```


此命令会列出所有活动连接及其对应的端口、协议和 PID（进程 ID）。

2. 查找特定端口

如果需要查找特定端口（如 8081）的占用情况，使用以下命令：

```bash
netstat -aon | findstr "8081"
```



输出中最后一列即为占用该端口的 PID。

3. 查看对应进程

根据 PID 查找具体的进程名称：

```bash
tasklist | findstr "PID号"
```



例如：

```bash
tasklist | findstr "9088"
```


4. 结束占用进程

如果需要释放该端口，可以通过以下命令结束对应进程：

```bash
taskkill /T /F /PID PID号
```



例如：

```bash
taskkill /T /F /PID 9088
```


注意事项

- 确保以管理员身份运行命令提示符，否则可能无法执行某些命令。
- 在结束进程前，请确认该进程是否为必要服务，以免影响系统或其他应用程序的正常运行。
